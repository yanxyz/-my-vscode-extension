# my-vscode-extension

我的 VSCode 扩展，添加的功能

- Reveal in Total Commander
- Open in Sublime Text
- Open in ConEmu
- Open in Integrated Terminal
- Copy Relative Path
- File: Create File，创建新文件，会创建目录
- File: Duplicate file
- File: Move file

开发

```sh
npm install -g vsce
yarn
code .
```

安装，见 `install.ps1`。
